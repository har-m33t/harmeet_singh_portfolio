# Quantum Entanglement Simulator

Interactive portfolio framed as a quantum entanglement simulator. Navigate entangled nodes (About, Projects, Work, Contact) connected by animated “quantum” links from a central observer (You).

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Optional assets

- **`public/profile-bw.jpg`** — Portrait for the About / Observer section.
- **`public/ambient-low.mp3`** — Optional ambient sound; enable in `app/page.tsx` with `useAmbientSound(true)`.

## Stack

Next.js (App Router), React, TailwindCSS, Three.js, React Three Fiber, Drei, Framer Motion, Zustand.
